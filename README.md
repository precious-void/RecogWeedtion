App that helps you recognise whether it`s plant or weed.

Just upload your photo and we`ll make decision to protect your garden!

## Installization
1. Clone repository
2. Install **Flask**, **Keras 2.1.6**
```
pip install flask
pip install keras==2.1.6
pip install opencv-python
```
3. Run Programm
```shell
cd RecogWeedtion 
python run.py
```
